#Laravel Project

## Service Layer
    -  é uma camada de serviço que encapsula a lógica de negócios relacionada a uma entidade. Seu papel principal é atuar como INTERMEDIARIA entre o CONTROLADOR e o REPOSITORIO que manipula os dados (geralmente interagindo com o banco de dados).

    - Mantém a lógica de negócios separada do Controller, o que torna o código mais Organizado e Modular.
